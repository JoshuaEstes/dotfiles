Command Line Aliases
====================
ls            = ls -hF --color
_             = sudo
q             = exit
..            = cd ..
....          = cd ..
-             = cd -
~             = cd ~
h             = history
v             = vim
ip            = This will display your external IP address.
simple-server = Start a web server in the path you are current in (http://localhost:8000)

gb  = git branch
gba = git branch -a
gc  = git commit -m
gca = git commit -am
gco = git checkout
gcb = git checkout - b
gd  = git diff
gpl = git pull
gp  = git push
gpo = git push origin
gr  = git remote -v
gss = git stash
gsl = git stash list
gsa = git stath apply
gsd = git stash drop

Cleanup Aliases
===============
cleanup-ds_store = find . -type f -name '*.DS_Store' -ls -delete
cleanup-svn      = find . -type d -name '*.svn' -ls -delete

Command Line Functions
======================
genpass [length] = Generates a password, first argument is the length. Default length is 16 characters

VIM Plugins
===========

F1       = Open NERDTree
F2       = Open Tagbar
F3       = List snippets (must be in insert mode)
ctrl + l = Clear highlighted search results
ctrl + p = Insert phpDoc block
